## 1. アカウントティア

ティアは **3つのみ**。シンプルに保つ。

### 未登録（ログインなし）

会員登録しなくても誰でもアクセスできるページ。

- Level Check
- Resources Library
- Chat'n'Chill 参加案内（/community）
- Life Balance Assessment
- Servicesページ

### Free（無料会員登録後）

| 項目 | 内容 |
|------|------|
| 条件 | メールアドレスで無料登録 |
| 料金 | 無料 |

**未登録で見れるページに加えて：**
- Community Board（一部閲覧のみ）
- 公開予約ページ（/booking）へのアクセス

### Premium

| 項目 | 内容 |
|------|------|
| 条件 | 月額課金 or 上位契約で自動付与 |
| 料金 | ¥500/月（Stripe サブスクリプション） |
| 自動付与 | GOWMベース契約者、Personal契約者は追加課金なしでPremium機能全開放 |

**Free機能に加えて：**
- AI Workbook
- Voice Lounge（2026年6月〜）
- 翻訳機能
- Community Board（Personal専用板以外すべて）
- Lesson Archive（閲覧）

### Personal

| 項目 | 内容 |
|------|------|
| 条件 | Sayakaが管理画面から手動で切り替え |
| 料金 | サイト上での課金なし（契約は別途） |
| 対象 | 年間契約の生徒、GOWM伴走セッション購入者 |

**Premium機能に加えて：**
- サイト内予約機能（ダッシュボードから直接予約）
- Lesson Archive（自分専用の記録）
- Community Board（全板アクセス）

## 2. ティア切り替えロジック

```
Free → Premium: Stripeサブスク開始で自動昇格
Premium → Free: サブスク解約で自動降格
Free/Premium → Personal: Sayakaが管理画面から手動切り替え
Personal → Premium: Sayakaが管理画面から手動切り替え
```

**重要：** Personal昇格は自動化しない。Sayakaが契約状況を確認して手動で行う。

## 4. 予約システム

### ① 公開予約ページ（/booking）

**アクセス条件**: 無料会員登録必須（CTAとして機能）

**基本仕様：**
- 全枠一律60分
- カレンダーから空き枠を選んで予約
- Stripe決済（¥10,000）
- クーポン入力欄あり → 価格を上書き

**クーポン種別：**

| クーポン名 | 適用後価格 | 条件 | 用途 |
|-----------|-----------|------|------|
| 初回クーポン | ¥2,000 | 1人1回限り | 新規体験 |
| Mocca Group | ¥1,100 | 週1回まで | リゾバ向けB2B |
| 無料相談 | ¥0 | Sayakaが個別発行 | ワーホリ相談等 |

**クーポン実装方法：** サイト独自のクーポンシステム。
- `booking_coupons` テーブルでクーポンを管理
- ユーザー紐付けクーポン（GOWM伴走、調整等）→ 予約時に自動適用、コード入力不要
- コード入力式クーポン（Mocca Special等）→ 予約フォームにコード入力欄
- クーポンで ¥0 の場合は Stripe 決済をスキップ
- クーポンで割引の場合は割引後価格で Stripe 決済

### ② ダッシュボード内予約（Personalティアのみ表示）

**アクセス条件**: Personalティアのユーザーのみ

**基本仕様：**
- 90分枠
- クーポン不要、決済不要（契約内）
- ログイン後ダッシュボードに「予約する」ボタンが見える
- Freeの人、Premiumの人には表示しない

## 5. 出演者専用掲示板

**ティアに紐づけない。合言葉（パスワード）で開放。**

### 仕様

- Community Board内に鍵付きカテゴリとして作成
- 初回アクセス時にパスワード入力 → 以降はアクセス可
- パスワードはSayakaが出演者に個別に伝える
- どのティアのユーザーでもパスワードがあればアクセス可

### パスワード管理

- Sayakaが管理画面からパスワードを設定・変更できる
- 定期的に変更可能（必要に応じて）

## 6. ワークブック進捗管理

AI Workbook・カスタムワークブック（GOWM教材含む）の学習進捗をサイト上で管理。

### 対象

- AI Workbook（Premium会員が自動生成）
- カスタムワークブック（¥3,000 単品購入 / GOWM教材）

### 機能

| 機能 | 内容 |
|------|------|
| 日次チェック | 各日のレッスンに完了マーク |
| 進捗バー | 完了日数 / 全体日数の視覚表示 |
| 連続学習日数 | ストリーク表示（モチベーション維持） |
| ダッシュボード | 自分のワークブック一覧 + 進捗サマリー |

### カスタムワークブックの登録フロー

```
Sayakaがヒアリング → 教材作成
  ↓
adminがワークブックを生徒のアカウントに紐付け
  ↓
生徒の /workbook に表示される
  ↓
日次コンテンツを閲覧 → 完了マーク → 進捗更新
```

### 既存DBとの関係

- `workbooks` テーブルの `days_completed` カラムで完了日数を管理（既存）
- `workbook_days` テーブルに日次コンテンツ格納（既存）
- カスタムワークブック用に `workbooks.source` カラム追加（'ai' | 'custom'）を検討
- admin からワークブックを特定ユーザーに割り当てる機能を追加

## 7. LINE無制限サポート（単品販売）

LINEでの個別質問・学習サポートを月額で提供。サイト外（LINE上）で実際のサポートを行うが、契約管理はサイト上で行う。

### 基本仕様

| 項目 | 内容 |
|------|------|
| 価格 | ¥5,000/月 |
| 種別 | Stripe サブスクリプション |
| 対象 | 誰でも購入可（ログイン必須） |
| 解約 | Stripe Billing Portal から |

### 販売パターン

| パターン | 価格 | 備考 |
|----------|------|------|
| LINE無制限 単体 | ¥5,000/月 | 質問し放題 |
| ワークブック + LINE無制限 | ¥3,000 + ¥5,000/月 | 教材納品後にLINEサポート開始 |
| GOWM オプション | ¥5,000/月 | GOWM契約内で追加 |
| Personal 契約内 | ¥0 | Personal に含まれる |

### サイト上の表示

- ダッシュボードに「LINE無制限サポート：有効 / 未契約」バッジ
- Servicesページに独立カードとして掲載
- ワークブック購入後のサンクスページで「LINE無制限を追加しませんか？」のアップセル導線
- admin画面で契約中ユーザーの一覧表示

### 実装

- Stripe に月額商品として作成
- `subscriptions` テーブルで管理（既存の仕組みを流用）
- `product_type: 'line_support'` で区別

## 10. GOWM出演コンサル パッケージ構造（参考）

サイト内で自動販売するのではなく、案内ページ + お問い合わせで契約。

### 初月（¥20,000）

- ガチヒアリングセッション（90分）
- フルカスタム30日教材
- GOWMサイトPremium機能 全開放（追加課金なし）
- 出演者専用掲示板アクセス（合言葉付与）
- リハーサルセッション 1回（擬似インタビュー、タイミングは本人が決定）

### 2ヶ月目以降（¥5,000/月）

- 新規30日教材（フルカスタム更新）
- Premium機能 継続
- 出演者掲示板 継続

### オプション（どの月でも追加可）

| オプション | 価格 | 備考 |
|-----------|------|------|
| 伴走セッション 90分 | ¥7,500/回 | 追加時 → Personalティアに手動昇格 |
| LINE無制限 | ¥5,000/月 | 個別質問し放題 |

### 期間

- 月単位（1ヶ月から）
- 本人が継続/終了を決定

### 価格シミュレーション

| パターン | 期間 | 月額内訳 | 合計 |
|----------|------|----------|------|
| 中上級者（1ヶ月完結） | 1ヶ月 | ¥20,000 | ¥20,000 |
| 初級（3ヶ月・自走） | 3ヶ月 | ¥20,000 + ¥5,000×2 | ¥30,000 |
| 初級（3ヶ月・セッション月2+LINE） | 3ヶ月 | 初月¥40,000 + 2-3月¥25,000×2 | ¥90,000 |

### 撮影プロセス

```
ベース契約開始
  ↓ 教材で自走 + 掲示板で質問
  ↓ 必要なら伴走セッション追加
  ↓ 本人が「準備できた」と判断したら
リハーサル（擬似インタビュー）
  ↓ 本人がreadyと感じたら
撮影本番
```

リハーサルで「まだだな」と本人が感じた場合 → 追加月の教材 or セッションで準備継続。判定ではなく、本人が体感して決める。
